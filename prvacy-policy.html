SafeGuard Bot - Privacy Policy

SafeGuard is a multitool bot focused on security for Discord servers. This Privacy Policy describes how we collect, use, and share information when you use the bot.

1. Information We Collect

  • Server Information: We collect server details such as server name, roles, channels, and user lists to enable bot functionality.
  • User Information: We collect user IDs and usernames to manage features like verification, moderation, and security commands. We may also store data on flagged users marked as scammers in our database.
  • IP Verification: If enabled by the server admin, we may use IP-based verification to enhance security. IP data is securely processed and stored only for verification purposes.

2. How We Use Information

  • Provide features like moderation, security checks, and automated role management based on server admin configurations.
  • Ensure server safety by performing bot and raid prevention verification checks.
  • Allow server admins to access commands like /ip-ban when IP verification is enabled.

3. Data Security and Retention

We take data security seriously and store user data in a secure database with limited access. Data collected for verification and security purposes is retained only as long as needed to provide these services.

4. Sharing of Information

We do not share collected data with third parties, except as necessary to comply with legal obligations.
